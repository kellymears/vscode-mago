# Code Actions

When mago reports an issue with an available fix, the extension provides VS Code code actions — the lightbulb menu that appears when you hover over a diagnostic.

## Quick Fixes

For each fixable diagnostic, the extension creates a code action that applies the fix directly in the editor. The action label indicates the safety level:

<Table
  headers={["Label", "Meaning"]}
  rows={[
    [<strong>Fix: rule-code</strong>, "Safe fix — guaranteed to preserve behavior"],
    [<strong>Fix (potentially unsafe): rule-code</strong>, "May change behavior in edge cases"],
    [<strong>Fix (unsafe): rule-code</strong>, "May change behavior — review carefully"],
  ]}
/>

Safe fixes are marked as **preferred**, so they appear first in the lightbulb menu and can be applied with the quick-fix keybinding.

## Fix Operations

Mago provides three types of fix operations, all translated to VS Code workspace edits:

- **Replace** — Replace a range of text with new text
- **Insert** — Insert text at a position
- **Delete** — Remove a range of text

## Fix All

Use the command palette commands to batch-apply fixes:

- **Mago: Fix File (Safe)** — Applies all safe fixes by running `mago lint --fix`
- **Mago: Fix File (All Including Unsafe)** — Applies all fixes including unsafe ones (with confirmation dialog)

After applying fixes, diagnostics are automatically re-run to update the Problems panel.

## Safe vs Unsafe

Mago classifies every fix with a safety level:

- **Safe**: The fix is semantically equivalent. Example: replacing `==` with `===` where types are already compatible.
- **Potentially Unsafe**: The fix *usually* preserves behavior but may not in all edge cases. Example: adding `declare(strict_types=1)`.
- **Unsafe**: The fix changes behavior. Example: removing unreachable code.

The extension respects these classifications:
- Quick fix code actions show the safety level in the label
- The "Fix File (Safe)" command only applies safe fixes
- The "Fix File (All Including Unsafe)" command requires explicit confirmation
